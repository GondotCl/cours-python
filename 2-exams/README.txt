pour ouvrir les fichiers Jupyter (.ipynb) : 

sur GitHub : cliquer sur le fichier maya.ipynb et cliquer sur le bouton en haut à gauche pour le télécharger.

aller sur ce site : 
https://jupyter.org/try-jupyter/lab/
puis, dans l'onglet à gauche, cliquer sur l'incône pour import (flèche vers le haut) et sélectionner le fichier voulu.

une fois importé, ouvrir le fichier maya.ipynb en double cliquant dessus dans la liste de fichiers à gauche.

faire Shift+L pour afficher les numéros des lignes de code Python.
